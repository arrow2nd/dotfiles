# agent-browser でのブラウザ操作

ブラウザでの動作確認には `agent-browser` を使用する。

操作方法は CLI に内蔵されている公式ガイドを参照すること（このファイルに転記しない。CLI 側が常に最新のため）。

```bash
agent-browser skills get core --full  # 基本的な使い方（最初に必ず読む）
agent-browser skills list             # 特殊用途のガイド一覧（Electron, Slack など）
```
