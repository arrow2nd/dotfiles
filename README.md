# dotfiles

## 環境

- Arch Linux / macOS (10.15 Catalina ~)
- git 導入済み

## 中身

### エディタ

- nvim
  - dein
- vsvim

### ターミナル

- kitty
- zsh
- sheldon

### ツール

- joplin (GUI / CLI)
- nekome
- twnyan

## フォント

[PlemolJP (Nerd Fonts)](https://github.com/yuru7/PlemolJP)

## 導入メモ

```
git clone git@github.com:arrow2nd/dotfiles.git
# or
git clone https://github.com/arrow2nd/dotfiles.git

cd dotfiles

# 初期設定
./install.sh

# 各環境に合った方法で言語等をインストール
```
